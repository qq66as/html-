HTML基础
	1、超文本标记语言。由标记构成，标记是预定义标记
	结构：
		<html>-----声明网页的标记
			<head>-----头部信息
				<title>标题</title>
			</head>
			<body>
				网页显示的主体
			</body>
		</html>	
	2、标记的使用规则：
		1、标记不区分大小写
		2、标记一般成对出现，由开始标记和结束标记构成
		3、标记可以嵌套使用，但不能交叉使用

	3、属性的使用：
		概念：可以控制标记的显示形式
		使用位置：在开始标记中使用
		格式：<开始标记 属性名=“属性值” 属性名=“属性值“ >
		不同的属性控制的样式不一样，不同的标记属性不尽相同
		

	4、网页的编辑
		1、使用记事本编辑网页
			1-1、新建文本文档
			1-2、修改后缀名为html或者htm
			1-3、用记事本打开文件，开始编辑网页内容
		2、使用工具开发
			dreamWeaver
			更改默认字体大小：编辑---首选参数--字体---代码视图
	5、标题标记：<h1>~<h6>
		特点：加粗显示，前后自动换行，属于块级标记
		属性：align  控制文本内容的对齐方式
			取值：left(左对齐，默认的对齐方式、缺省值)|center(居中)|right(右对齐)
	6、段落标记：<p>
		特点：把内容分为一段显示，前后自动换行，属于块级标记
		属性：align  控制文本内容的对齐方式
			取值：left(左对齐，默认的对齐方式、缺省值)|center(居中)|right(右对齐)
	7、换行标记：<br> 该标记是单标记，不需要结束标记
		代码中的换行在页面被解析为空格
		格式：内容<br />
		空格：&nbsp;
	8、字符格式化标记
		<b> 加粗显示
		<i> 斜体
		<u> 下划线
		<sub> 下标
		<sup> 上标
		<s>  删除线
	9、水平线标记：<hr> 单标记
		特点：在页面添加水平线
		属性
		width 控制水平线的长度，取值像素值类似于绝对取值，取值百分数，随浏览器窗口大小调整
		size  控制水平线的粗细
		noshade 没有阴影
		align 控制水平线的对齐方式，取值left|center(缺省值)|right
		color 控制水平线的颜色，取值可以是十六进制的颜色代码或者英文颜色名
	10、字体控制标记：<font>
		特点：控制字体的显示效果
		color 控制文本的颜色
		size 控制文本的大小，取值绝对值1~7，数字越大字体越大，默认字体3，相对取值-2~+4，相对默认字体加大或缩小
		face 控制文本的字体，取值可以是多个值，值与值之间使用逗号分割，从左往右依次匹配字体的字体，如果找到支持的字体，则使用，不再往后匹配，假如字体都不支持，则使用默认字体


	11、图片标记：<img> 单标记
		特点:在页面中添加图像
		属性：src  要插入图像的位置(图像所在的路径)，绝对路径，表示图像在电脑上的真实位置
			相对路径，从网页自身出发
			align 控制文本和图片的对齐方式
			取值 bottom 文本的底部和图片的地步对齐
			        middle 文本的底部和图片的中间对齐
			       top 文本的顶部和图片的顶部对齐
			       absmiddle 文本的中间和图片的中间绝对对齐
				left  图片在左，文本在右
				right 图片在右，文本在左
		alt 当图像无法加载或者显示时间过长时的替换文字
		height 控制图像的高度，取值为像素值或者百分数
		width 控制图像的宽度，取值像素值或者百分数
		border 控制图像的边框，取值为像素值
	12、插入多媒体<embed>
		特点：在页面中添加多媒体文件
		属性：src  引入多媒体文件的路径
			width 多媒体播放器的宽度
			height 多媒体播放器的高度
			autostart 是否自动播放
	13、无序列表：<ul>，需要配合<li>标记使用
		格式：	<ul>---声明无序列表
				<li></li>
				<li></li>
				.......
			</ul>
		一个<li>表示一个项目或者选项
		<ul>的属性type控制列表的项目符号，取值circle|disc(缺省值)|square	
		<li>的属性type控制当前选项的项目符号，取值circle|disc(缺省值)|square	
	14、有序列表：<ol>，需要配合<li>标记使用
		格式：<ol>---声明有序列表
				<li></li>
				<li></li>
				.......
			</ol>
		<ol>的属性
			type 控制列表的符号样式，取值1(缺省值)|a|A|i|I
			start 控制列表的起始位置，取值是正整数
		<li>的属性
			type 控制当前选项的符号样式，取值1(缺省值)|a|A|i|I
			value 控制当前选项的位置，取值是整数，该取值会对同一个列表中，同级的<li>标记顺序产生影响
	15、定义列表：<dl>，需要配合<dt>、<dd>标记使用
		格式：<dl>----声明定义列表
				<dt></dt>---类似于标题
				<dd></dd>----类似于内容
				.....
				<dt></dt>
				<dd></dd>
				....
			</dl>
	16、超链接：<a>
		特点：通过路径链接到指定的资源
		属性：
		href  目标资源的路径，绝对路径和相对路径
		target 指定目标资源的打开方式，取值_self(在自身页面打开，默认打开)|_blank(打开新的浏览器窗口)|_top|_parent
		name 给锚点起名字
		内部锚点
		格式：<a name="锚点名">锚点的位置</a>，配合<a href="#锚点名">跳转到指定锚点</a>
		外部锚点
		格式：<a name="锚点名">锚点的位置</a>，配合<a href="锚点所在页面的路径#锚点名">跳转到指定锚点</a>
		电子邮件链接：<a href="mailto:邮箱地址">提示</a>

	17、表格
	格式：	<table>----声明表格
			<caption>表格的标题</caption>
			<tr>----声明行
				<td></td>-----声明列
				。。。。
			</tr>
			<tr>
				<td></td>
				。。。。
			</tr>
			。。。。
		</table>
	所有表格的内容全部书写在<table>标记之内
	<table>的属性：
		border  控制表格的边界，取值为像素数
		align 控制表格的对齐方式，取值left(缺省值)|center|right
		bgcolor 设置表格的背景色
		cellpadding 单元格内容和边界之间的距离，取值为百分数或者像素数
		cellspacing 单元格与单元格之间的空白，取值百分数或者像素数
		frame 控制表格外层四个边框是否显示
			取值：void 不显示，缺省值
				above 仅显示上边框
				below 仅显示下边框
				hsides 显示上下边框
				lhs 仅显示左边框
				rhs 仅显示右边框
				vsides 显示左右边框
				box、border 显示四个边框
		rules 控制表格的内部分割线
			取值：none 不显示分割线，缺省值
				rows 显示行分割线
				cols 显示列分割线
				all 显示所有分割线

		边界的控制border属性使用更为广泛
		height 设置表格的高度，取值为像素或者百分数
		width 设置表格的宽度，取值为像素或者百分数
		background 设置表格的背景图
		背景图背景色同时设置，背景图生效

	<tr>的属性：
		align  控制整行文本的水平对齐方式，left(左对齐)|center|right
		valign 控制整行文本的垂直对齐方式，top|middle(缺省值)|bottom
		bgcolor 设置整行的背景色
		background 设置整行的背景图
	<td>的属性
		align  控制单元格文本的水平对齐方式，left(左对齐)|center|right
		valign 控制单元格文本的垂直对齐方式，top|middle(缺省值)|bottom
		bgcolor 设置单元格的背景色
		background 设置单元格的背景图
		colspan 合并列
		rowspan 合并行
	

	表单
		表单标记：<form>
		表达域：<input>单标记
			不同功能表单域type属性的值不同
			属性：type
			取值：	text	单行文本
				password	密码
				checkbox	多选框
				radio	单选框
				submit	提交
				reset	重置
				button	自定义按钮
				file	文件 
			在单行文本和密码框中maxlength属性可以限制最大字符输入数
			在多选框和单选框中使用checked=”checked“可以让该选项默认被选中
			value 给表达域设置初始值

		多行文本：<textarea>
			属性：rows 设置多行文本的行数
				cols 设置多行文本的列数
		下拉菜单：<select>,需要配合<option>使用
			默认显示第一个选项，如果想要更给可以使用selected=“selected”

	框架
		声明框架：<frameset>，不能和body共存
		属性：	cols 把浏览器窗口纵向分割 
			rows 把浏览器窗口横向分割
			noresize 固定框架的大小
			frameborder 设置框架是否显示边框，取值为0不显示，为1显示
		cols和rows一般不同时使用，要想分割多个框架，可以使用框架的嵌套
		cols和rows可以同时取多个值，值与值之间使用逗号分割，注意不要有空格
		值的个数决定了框架的个数
		框架：<frame>，表示一个框架
		属性：	src 引入目标资源的路径
			noresize 固定框架的大小
			frameborder 设置框架是否显示边框，取值为0不显示，为1显示
			scrolling 控制框架是否加入滚动条，取值yes|no|auto(缺省值)
			marginwidth 控制框架内容距离左右边框的空白
			marginheight 控制框架内容距离上下边框的空白
			name 给框架起名字,需要配合超链接的target属性使用
	内嵌框架：<iframe>
		在<body>中引入内嵌框架
		属性：src 引入目标资源的路径
			frameborder 设置框架是否显示边框，取值为0不显示，为1显示
			scrolling 控制框架是否加入滚动条，取值yes|no|auto(缺省值)
			marginwidth 控制框架内容距离左右边框的空白
			marginheight 控制框架内容距离上下边框的空白
			name 给框架起名字,需要配合超链接的target属性使用
			height 设置内嵌框架的高度
			width 设置内嵌框架的宽度
			align 设置内嵌框架的对齐方式（包含垂直方向和水平方向）
